Adafruit STMPE610 data sheet can be found here: http://www.adafruit.com/products/1571


- On the Arduino IDE, go to Sketch -> Include Library -> Manage Library -> Search for ‘STMPE610’ -> Install
- Restart the IDE

- Pin connections are:
SDA/CS	-> A4
SCL 	-> A5
Mode	-> GND (on STMPE610)
GND	-> GND (on Arduino)
Vin	-> 5V (on Arduino)
